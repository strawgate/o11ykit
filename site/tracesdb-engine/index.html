How It Works

o11ytracesdb stores OpenTelemetry spans in a columnar layout optimized for the browser. Bloom filters accelerate trace ID lookups. Nested set encoding enables O(1) structural queries. Dictionary encoding compresses repeated span names and attribute keys.
